## Project 1: Domain Controller Compromise Investigation

### Project Overview

This project documents a simulated SOC and DFIR investigation involving:

- Unauthorized RDP access
- Malicious PowerShell execution
- Windows Defender and Firewall disablement
- Registry, scheduled-task, and WMI persistence
- Mimikatz credential-access activity
- Lateral movement to a domain controller
- NTDS database extraction
- Sensitive-data collection
- Possible DNS and web-based exfiltration
- Event-log and PowerShell-history clearing

### Skills Demonstrated

- Microsoft Defender Investigation
- Windows event analysis
- Incident timeline development
- MITRE ATT&CK mapping
- SOC and DFIR report writing
- Impact assessment
- Evidence documentation
- Containment and recovery recommendations
